# 💼 Annual Salary Prediction

A Machine Learning web application that predicts an employee's annual salary based on professional and personal attributes.

## Features

- Age
- Education
- Experience
- Department
- Job Level
- City
- Company Tenure
- Projects Completed
- Skill Score

## Model

Gradient Boosting Regressor

## Technologies

- Python
- Streamlit
- Scikit-Learn
- Pandas
- NumPy

## Run

```bash
streamlit run app.py
```
